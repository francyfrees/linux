## 文件内容查阅

```

cat 由第一行开始显示文件内容
tac 从最后一行开始显示，可以看出 tac 是 cat 的倒着写!
nl 显示的时候，顺道输出行号!
more 一页一页的显示文件内容 
less 与 more 类似，但是比 more 更好的是，他可以往前翻页! 
head 只看头几行
tail 只看尾巴几行
od 以二进制的方式读取文件内容!

```
